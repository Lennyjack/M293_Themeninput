HTML-Formulare

Formulare werden mit dem <form>-Tag erstellt und enthalten verschiedene Eingabeelemente.









Übersicht der Input-Elemente

  • Textfelder: <input type="text">
  • Passwortfeld: <input type="password">
  • E-Mail: <input type="email">
  • Checkbox: <input type="checkbox">
  • Radiobutton: <input type="radio">
  • Datei-Upload: <input type="file">
  • Dropdown-Menü: <select><option>

Daten von Formularen senden

Daten können mit den Methoden "GET" oder "POST" gesendet werden. Diese werden im <form>-Tag definiert.

Get wird dafür verwendet um daten von einem server abzurufen.

Post Dient dazu si cher daten an den server zu senden.

Navigation umsetzen

Navigationen können mit dem <nav>-Tag umgesetzt werden. Ein einfaches Beispiel: 

<nav>
<ul>
    <li><a href="#ziel">Link</a></li>
</ul>
</nav>

Links und Anker

Verweise in HTML werden mit dem <a>-Tag gesetzt. Ein Beispiel für einen Anker: 

<a href="#ziel">Gehe zu Ziel</a>
<div id="ziel">Hier ist das Ziel</div>

CSS Grundlagen

Was ist CSS?

CSS (Cascading Style Sheets) ist eine Sprache, die das Design von HTML-Dokumenten steuert.

Selektoren und Syntax

Die Syntax einer CSS-Regel besteht aus einem Selektor und einer Deklaration: 

selektor {
eigenschaft: wert;
}

Implementierungsvarianten

  • Inline: Direkt im HTML-Element mit dem "style"-Attribut
  • Intern: Innerhalb eines <style>-Tags im HTML-Dokument
  • Extern: Verlinkung zu einer separaten CSS-Datei

CSS-Reset

Ein CSS-Reset wird verwendet, um Browser-Standardstile zu entfernen, damit alle Browser von einer gleichen Basis ausgehen.

CSS-Einheiten

  • Absolute Einheiten: px, cm, mm
  • Relative Einheiten:
    em (1em=16px)
    rem (Einheit ist relativ zur Schriftgröße)
    % (Einheit ist relativ zur Größe ihres Elternelements.)
    vw/vh (Diese Einheit ist relativ zur Breite des Ansichtsfensters)
